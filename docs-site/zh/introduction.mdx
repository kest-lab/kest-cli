# Kest 文档站

Kest 是一个面向 AI 工作流的 API 测试平台，核心由终端优先的 CLI 和协作式 Web 控制台组成。

本站分为两层：

- `文档` 负责说明认证、错误模型、CLI Sync 和常见工作流。
- `API 参考` 由 Kest 的公开 OpenAPI 契约生成，是接口、请求体和响应结构的最终来源。

## 你可以用 Kest 做什么

- 创建和管理 API 项目
- 从本地 CLI 工作流导入或同步 API 规格
- 管理环境、集合、请求与示例
- 生成、运行并查看测试用例
- 通过项目成员角色进行协作

## 本版 Mintlify 覆盖范围

本版文档站严格以当前后端代码里真实暴露的 API 为准，覆盖：

- 认证与用户
- 项目与 CLI Sync
- API Specs
- Environments
- Categories
- Test Cases
- Members 与 Permissions
- Collections、Requests、History
- Workspaces
- Health 与 System 元数据

仓库中原有的 `/docs/api` 仍然可以作为素材来源，但本站优先遵循代码路由和 `/docs-site/openapi` 下的新 OpenAPI 文件。
